To open ".ipynb" files on CoLab:

1.Please click "File" button on top left.

2.Find "upload notebook" button.

3.The window will pop up and let you upload notebook.

4.Upload "Final_SentimentAnalysis_CellphoneReviews.ipynb" file, then the code should be showed on the screen.

------------------------------------------------

To run a demo:

1. Click the arrow button at square bracket.

2. Click "Choose Files" button to upload the datasets.

3. Choose "cellphone_3w.csv" to upload.

4. Click each of arrow buttons on cells sequentially.

5. The results should be showed below cells.
